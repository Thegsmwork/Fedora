## Installation

Copy and paste this command to Termux :

```
pkg update && pkg upgrade
```

1. Fedora 36.1.5
* Install Fedora

```
pkg install wget -y && wget https://raw.githubusercontent.com/wahasa/fedora/main/install-fedora.sh && chmod +x install-fedora.sh && ./install-fedora.sh
```

2. Start Fedora

```
fedora
```

3. Stop Fedora

```
exit

```
